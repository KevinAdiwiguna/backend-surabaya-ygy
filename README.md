
## Installation

Dillinger requires [Node.js](https://nodejs.org/) v16+ to run.

## Setup The Project

Open Git bash on Desktop

```bash
  git clone https://github.com/KevinAdiwiguna/backend-surabaya-ygy.git
```

```bash
  cd backend-surabaya-ygy
```

```bash
  npm i 
```
```bash
  npm run dev 
```
